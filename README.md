# Vue 3 + Vite

## Creacion del proyecto
```
create vite@latest
```
```
npm install
```

## Corremos el proyecto:
```
npm run dev
```

## Instalacion del router
```
npm install vue-router@4
```
## Instalacion del SweetAlert2
```
npm install sweetalert2
```
## Run para mostrar el host
```
npm run dev -- --host 
```
